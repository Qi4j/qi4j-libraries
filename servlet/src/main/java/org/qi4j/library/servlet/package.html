Qi4j Library - Servlet

This library provide the necessary mechanism to bootstrap a Qi4j Application in a Servlet container plus some facilities.

Logging

The SLF4J logger used by this library is named "org.qi4j.library.servlet".

Application Bootstrap

Extends {@link AbstractQi4jServletBootstrap} to easily bind a Qi4j {@link Application} activation/passivation to your webapp lifecycle.

Use {@link Qi4jServletSupport#application(javax.servlet.ServletContext)} to get a handle on the {@link Application} from the {@link ServletContext}.

Facilities

{@link Qi4jServlet} and {@link Qi4jFilter} respectively provide base class for easy access to the {@link Application} from the {@link ServletContext}.
